API Documentation

Introduction

The Text Analysis API provides natural language processing capabilities for sentiment analysis, entity extraction, and text summarization. This API is designed to help developers integrate advanced text analysis features into their applications with minimal effort.

Authentication

All API requests require authentication using an API key. Include your API key in the request headers as follows:

X-API-Key: your_api_key_here

Rate Limits

The API has the following rate limits:

  • Free tier: 100 requests per day
  • Basic tier: 1,000 requests per day
  • Premium tier: 10,000 requests per day
